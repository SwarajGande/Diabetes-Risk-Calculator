# Diabetes-Risk-Calculator
**This is an End-to-End Data Science project which can predict whether a person has diabetes, or not, based on information about the patient such as blood pressure, body mass index (BMI), age, etc.**
## Model Accuracy : 98.25% ##
## About The Model : ##
- Trained the Machine Learning Model with features required like Glucose Level, BMI,Pregnancies and diabetes pedigree function 
- Used Random forest Algorithm for classification
